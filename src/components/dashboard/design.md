# Mind365 Design System — 皮质手账风格

## 核心美学
笔记本实物感。不是扁平 App，是一本真实存在的手账。
每个细节都服务于"这是我的私人本子"的感受。

## 色彩系统
```css
--cover-dark:    #5C3D22;  /* 皮质封面 */
--cover-mid:     #7A5538;  /* 封面高光 */
--page-left:     #F0E8D8;  /* 左页（暖） */
--page-right:    #F5EFE2;  /* 右页（略冷） */
--page-edge:     #E2D5C0;  /* 页面边缘/书脊 */
--ruled-line:    #DDD3C0;  /* 横线颜色 */
--text-primary:  #3A2E22;  /* 主文字 */
--text-secondary:#8A7260;  /* 次级文字/标签 */
--stamp-color:   #9A7A5A;  /* 印章颜色 */
--tag-bg:        #EAE0CC;  /* 标签背景 */
```

## 字体
- 正文：Noto Serif SC（衬线，手账质感）
- 英文标题：Playfair Display 或 DM Serif Display
- 小标签/页脚：系统等宽字体
